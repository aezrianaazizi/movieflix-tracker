MovieFlix Tracker

A Netflix-style movie tracking web application developed for IMS566 Advanced Web Design Development and Content Management individual project.

Features
- Login Authentication
- Responsive Navigation Bar
- Dashboard with Movie Statistics
- Chart Visualization using Chart.js
- Movie List
- Watchlist Table
- Favorites Section
